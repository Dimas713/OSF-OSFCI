HPE Open Source Firmware - Continuous Integration Platform

Welcome to the HPE Open Source Firmware Continuous Integration Platform. This tool is being provided as a resource to the Open community to accelerate the general availability of open source firmware on HPE platforms. Feedback on the CI platform or HPE's support for Open firmware is appreciated.

 

OSF-CI Features

OpenBMC

Dynamically load a standard or custom prebuilt OpenBMC FW payload via drag and drop

 

 

LinuxBoot

Dynamically load the standard HPE BIOS, build and load LinxBoot directly from a github repo and branch or drag or drop a prebuilt FW payload

 

 

Boot Process

Load the desired OpenBMC and HPE/Linuxboot FW images

 

 

Connect power to host.  

Once boot is complete an interactive console (root/0penBmc) is available

 

 

Launch OpenBMC web session.  

 

Disconnect power from host and end session.
