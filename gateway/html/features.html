OSF-CI Features

OpenBMC

Provides the capability to dynamically load a standard or custom prebuilt OpenBMC FW payload via drag and drop

 

 

LinuxBoot

Provides the capability to dynamically load the standard HPE BIOS, build and load LinxBoot directly from a github repo and branch or drag or drop a prebuilt FW payload

 

 

Boot Process

Load desired OpenBMC and HPE/Linuxboot FW images

 

Connect power to host.  

Once boot is complete an interactive console (root/0penBmc) is available

 

Launch OpenBMC web session.  

 

 

Disconnect power from host and end session.
